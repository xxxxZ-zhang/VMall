## 技术栈

vue + vue-router + vuex + axios + vant-ui + less + betterScroll


## 项目运行

```
项目运行之前，请确保系统已经安装以下应用
1、node (10.0 及以上版本)
```

```
git clone https://github.com/bailicangdu/node-elm  

cd vmall

npm install

npm run serve

访问: http://localhost:8080

```
